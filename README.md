## Dependencies

* __Python__
* __Flask__
* __SQLAlchemy__
```bash
npm install
```

## Run the Server

From within the root directory first ensure you are working using your created virtual environment.

To run the server, execute:

```bash
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run --reload
```
